# DHT11-Thermometer-Alarm-System
Arduino project based around the DHT11 to monitor both humidity and temperature whilst consisting of alarm like functionalities. When the Temperature breaches 80 degrees, an LED and a buzzer will simultaneously go off until the temperature falls below the threshold.

Features:
- Real-time temperature and humidity monitoring
- 16x2 I2C LCD display output
- Visual alarm using LED
- Audible alarm using buzzer
- Manual disarm button (hold the button down)
- Serial monitor output for debugging
